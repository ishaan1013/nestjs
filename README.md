## Project Description

A basic Nest.js REST API following the MVC pattern, integrated with Prisma + PostgreSQL with Neon. It implements CRUD, rate limiting, logging, and excepton filtering.

## Installation + Running

```bash
$ npm install
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
